# ✅ **Fix Step-by-Step (Guaranteed Clean Reset)**

## **1. Remove `_posts` from the index**

Even if the folder isn’t there anymore, run:

`git rm -r --cached _posts`

If Git says “path not found,” try:

`git rm --cached _posts`



# **Option 2 — Make Repo B a submodule inside Repo A**

If you want Repo B to _live inside_ Repo A:

### In Repo A:

`git submodule add <repo-B-url> _posts`

Then Repo B _becomes_ the `_posts` folder.

This means:

- Repo B is independent
    
- Repo A references Repo B
    
- **BUT** Repo A no longer owns `_posts/` — Repo B does
    

This works if you don’t need `_posts` tracked by Repo A anymore.
